# SimpleBankingSystem

Backend learning project implementing a basic banking REST API using Spring Boot, MySQL, and JPA with CRUD operations tested via Postman.

# Spring Boot Banking REST API

This is a backend learning project built to practice REST API development using Spring Boot and MySQL.

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- Postman (for testing)

## Features
- Create bank accounts
- View account details
- Update account balances (deposit/withdraw)
- RESTful endpoint structure
- Database persistence using MySQL

## Purpose

The goal of this project was to understand:
- How REST APIs are structured
- How Spring Boot handles dependency injection and auto-configuration
- How to connect a Java backend to a MySQL database
- How to test endpoints using Postman
